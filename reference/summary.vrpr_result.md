# One-row summary of a result (tibble)

One-row summary of a result (tibble)

## Usage

``` r
# S3 method for class 'vrpr_result'
summary(object, ...)
```

## Arguments

- object:

  A
  [`vrp_solve()`](https://strategicprojects.github.io/vrpr/reference/vrp_solve.md)
  result.

- ...:

  Unused.

## Value

A one-row tibble with cost, feasibility, number of routes, iterations
and runtime.
