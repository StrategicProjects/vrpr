# Generate a random solution

Generate a random solution

## Usage

``` r
vrp_random_solution(problem_data, seed = 42L)
```

## Arguments

- problem_data:

  A
  [`vrp_problem_data()`](https://strategicprojects.github.io/vrpr/reference/vrp_problem_data.md).

- seed:

  Integer seed.

## Value

A `vrpr_solution` object.
